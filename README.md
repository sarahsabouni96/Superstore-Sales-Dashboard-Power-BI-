# Superstore-Sales-Dashboard-Power-BI-
This Power BI dashboard visualizes sales, profit, and customer distribution from the Superstore dataset. It helps identify product trends, regional patterns, and operational insights related to shipping and discounts.

## Dashboard Insights
- Sales and profit trends over time
- Product performance by category and sub-category
- Shipping mode comparison and delivery time
- Customer distribution across regions and states

## Goal
Identify trends, high-performing products, and opportunities to improve business operations.

## Tools Used
- Power BI
- Power Query
- DAX
